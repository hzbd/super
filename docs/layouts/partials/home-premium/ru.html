Безопасность и governance

plugins/security.so
  • Bearer-аутентификация на API-маршрутах
  • RBAC — admin, operator, viewer
  • Неизменяемый журнал аудита операций

Уведомления о событиях

plugins/notify.so
  • Webhook — Slack, DingTalk, Feishu, custom URL
  • Push при process_fatal и других событиях
  • Hot-reload через conf/notify.toml

Изоляция ресурсов

plugins/isolation.so
  • Лимиты CPU/памяти через cgroups v2
  • Hot-update лимитов без перезапуска процесса
  • CLI --cpu / --memory

🐧 Только Linux — недоступен на macOS, Windows и FreeBSD.
