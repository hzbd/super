Welcome to Project Super. This guide will help you install Super and run your first managed process in under 5 minutes.

Super is designed to be **dependency-free** and **secure by default** in OSS: loopback bind, fail-closed public exposure, and explicit opt-in when you need a network-facing API without the `security` plugin. See [Configuration — OSS security defaults](/docs/02-essentials/configuration#oss-security-defaults-fail-closed).

### In this section

*   [**Installation**](./installation): Install via binary, Docker, or Systemd.
*   [**Quick Start**](./quick-start): Write your first configuration and launch a service.
