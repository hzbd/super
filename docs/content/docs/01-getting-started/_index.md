Welcome to Project Super. This guide will help you install Super and run your first managed process in under 5 minutes.

Super is designed to be **dependency-free**. Whether you are running on a bare-metal Linux server, a Raspberry Pi, or inside a Docker container, Super provides a consistent and robust process management experience.

### In this section

*   [**Installation**](./installation): Install via binary, Docker, or Systemd.
*   [**Quick Start**](./quick-start): Write your first configuration and launch a service.
